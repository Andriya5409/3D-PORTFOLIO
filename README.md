# 3D-PORTFOLIO
MY PERSONAL INTERACTIVE PORTFOLIO

A modern, interactive, and visually immersive personal portfolio website built to showcase skills, projects, leadership experience, and achievements.

This portfolio blends **creative UI/UX**, **3D visuals**, and **smooth animations** to deliver a unique and engaging user experience.

---

## ✨ Features

### 🎯 Interactive UI

* Custom animated cursor with hover effects
* Smooth scrolling and section transitions
* Glassmorphism-inspired design

### 🎨 Visual Experience

* Neon-themed gradient design system
* Animated avatar with rotating rings
* Scroll-based reveal animations

### 🧠 3D Graphics

* Built using **Three.js**
* Dynamic particle field background
* Floating 3D rings reacting to mouse movement

### 🧩 Sections Included

* **Hero Section** – Introduction with animated elements
* **About** – Personal overview and traits
* **Skills** – Categorized tech stack with hover levels
* **Projects** – Highlighted work (EduGuide + upcoming projects)
* **Leadership** – Timeline of roles and impact
* **Achievements** – Awards and recognitions
* **Contact** – Interactive links + animated orbit UI

### 🎉 Easter Egg

* Triple-click on your name activates **Fun Mode**

  * Rainbow animations 🌈
  * Confetti burst 🎊
  * Dynamic visual effects

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **3D Graphics:** Three.js
* **Fonts:** Google Fonts (Space Mono, Syne, Inter)

## 🔧 Customization

You can easily personalize this portfolio:

* ✏️ Edit **text content** (name, bio, projects)
* 🎨 Modify colors using CSS variables in `:root`
* 🧠 Add more projects in the **Projects section**
* 🔗 Update contact links (GitHub, LinkedIn, Email)
* 📄 Replace the resume button with your actual resume link
